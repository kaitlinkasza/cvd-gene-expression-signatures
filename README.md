# Gene Expression Signatures of Cardiovascular Disease Risk

## Overview

This project investigates transcriptomic biomarkers
associated with cardiovascular disease using publicly
available gene expression datasets.

Analyses include:
- differential gene expression
- machine learning classification
- pathway enrichment
- biological interpretation

## Objectives

- Identify genes differentially expressed in CVD
- Build predictive transcriptomic classifiers
- Investigate biological pathways associated with disease risk

## Methods

- GEO transcriptomic datasets
- Differential expression analysis
- Multiple testing correction
- Random forest classification
- Pathway enrichment analysis
